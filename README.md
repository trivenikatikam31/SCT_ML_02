# Customer Segmentation using K-Means Clustering

## Objective
Group mall customers based on purchase behavior using K-Means clustering.

## Dataset
Mall Customer Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Steps Performed
1. Data Loading
2. Data Exploration
3. Feature Selection
4. Elbow Method
5. K-Means Clustering
6. Cluster Visualization
7. Customer Segmentation Analysis

## Results
Successfully divided customers into 5 segments based on annual income and spending score.
